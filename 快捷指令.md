# Audio Player 快捷指令（供 exe 调用）

exe 通过 **adb** 调用以下命令即可控制本 APK：打开应用、播放、暂停、继续播放、重新播放。

---

## 一、安装 APK（首次或更新时）

```bash
adb install -r <APK完整路径>
```

示例（按你本机路径替换）：

```bash
adb install -r E:\file\Jira\AI-Demo\apk\PlayerDemo\app\build\outputs\apk\release\app-release-unsigned.apk
```

- `-r`：已安装则覆盖安装。

---

## 二、打开 App 并直接播放

**作用**：启动 Audio Player，并自动开始播放；若已在后台则回到前台并开始/继续播放。

```bash
adb shell am start -a com.player.demo.PLAY -n com.player.demo/.MainActivity
```

---

## 三、暂停播放

**作用**：若正在播放则暂停；应用未启动或已暂停则无效果。

```bash
adb shell am start -a com.player.demo.PAUSE -n com.player.demo/.MainActivity
```

---

## 四、继续播放（暂停后恢复）

**作用**：若已暂停则从当前进度继续播放；正在播放或应用未就绪则无效果。

```bash
adb shell am start -a com.player.demo.RESUME -n com.player.demo/.MainActivity
```

---

## 五、重新播放（从头播放）

**作用**：把进度拉到 0:00 并开始播放；若应用未启动会先启动再从头播放。

```bash
adb shell am start -a com.player.demo.REPLAY -n com.player.demo/.MainActivity
```

---

## 六、仅打开 App（不自动播放）

**作用**：只打开主界面，不自动播放，用户可手动点播放。

```bash
adb shell am start -n com.player.demo/.MainActivity
```

---

## 七、exe 中如何调用

在 exe 里用 **Shell 执行** 上述 adb 命令即可，例如：

| 需求         | 执行的命令 |
|--------------|------------|
| 安装/重装 APK | `adb install -r <APK路径>` |
| 打开并播放   | `adb shell am start -a com.player.demo.PLAY -n com.player.demo/.MainActivity` |
| 暂停播放     | `adb shell am start -a com.player.demo.PAUSE -n com.player.demo/.MainActivity` |
| 继续播放     | `adb shell am start -a com.player.demo.RESUME -n com.player.demo/.MainActivity` |
| 重新播放     | `adb shell am start -a com.player.demo.REPLAY -n com.player.demo/.MainActivity` |
| 只打开界面   | `adb shell am start -n com.player.demo/.MainActivity` |

- 需保证设备已通过 USB 连接并开启 **USB 调试**，且本机已安装 **adb** 并加入 PATH。
- 多台设备时可用 `-s <设备序列号>`，例如：  
  `adb -s XXXXX shell am start -a com.player.demo.PLAY -n com.player.demo/.MainActivity`

---

## 八、Intent Action 汇总（供开发参考）

| Action                    | 说明             |
|---------------------------|------------------|
| `com.player.demo.PLAY`    | 打开并自动播放   |
| `com.player.demo.PAUSE`  | 暂停播放         |
| `com.player.demo.RESUME` | 继续播放（暂停后恢复） |
| `com.player.demo.REPLAY` | 打开并从头重新播放 |
| 包名 / Activity           | `com.player.demo` / `com.player.demo/.MainActivity` |
